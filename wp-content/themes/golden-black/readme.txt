== Description ==
Golden Black is a beutiful theme for blogs. This theme is mobile-first with off-canvas navigation. The layout comes with a right sidebar (optional on pages).

* Mobile-first, Responsive Layout
* Beautiful design
* Dark background and golden colors for text
* Custom Logo
* Right Sidebar
* Featured Images
* The GPL v2.0 or later license

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Golden Black in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize.
